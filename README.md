# Lead Management CRM

A scalable CRM MVP built using React and Firebase to manage leads across their lifecycle.

## Overview

This system allows users to capture leads, track their progress through a sales pipeline, assign ownership, schedule visits, and monitor activity through a timeline and dashboard.

## Key Features

* Lead creation and management
* Pipeline tracking (New → Closed)
* Visit scheduling
* Real-time updates using Firebase
* Activity timeline for each lead
* Centralized dashboard with recent activity

## Tech Stack

* React (Frontend)
* Firebase Firestore (Backend)

## Architecture

React UI connected to Firebase Firestore with real-time data synchronization.
